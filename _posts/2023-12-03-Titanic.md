### 3-1. Impute

```python
from sklearn.impute import SimpleImputer
```


```python
# mean: 평균값, median: 중앙값, most_frequent: 최빈값, constant: 특정값 -> fill_value=값
imputer = SimpleImputer(strategy='mean')
```


```python
imputer.fit(train[['Age', 'Pclass']]) # fit()을 통해 결측치에 대한 mean 학습을 진행
```
